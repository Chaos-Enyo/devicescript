# Adafruit Seesaw

The `@devicescript/seesaw` package provides basic support for Adafruit Seesaw. Adafruit Seesaw is an I2C protocol to communicate with multiple sensors and devices over a single wire.

-   [Documentation on adafruit.com](https://learn.adafruit.com/adafruit-seesaw-atsamd09-breakout/overview)

:::info

The package is under construction and is in need of community contributions to add support for the various devices that can be connected to the seesaw.

See https://github.com/microsoft/devicescript/discussions/421

:::

## Usage

Instantiate the driver

```ts
import { SeesawDriver } from "@devicescript/seesaw"

const seesaw = new SeesawDriver()
```
